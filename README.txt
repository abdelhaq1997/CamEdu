# لعبة الكلمات العربية - نسخة GitHub Pages + PWA

## الملفات
- `index.html`
- `sw.js`
- `manifest.webmanifest`
- `icons/`

## الرفع على GitHub
1. أنشئ مستودعًا جديدًا.
2. ارفع كل الملفات كما هي في جذر المستودع.
3. ادخل إلى **Settings > Pages**.
4. اختر **Deploy from a branch**.
5. اختر الفرع `main` والمجلد `/root`.
6. انتظر حتى يظهر رابط GitHub Pages.

## ملاحظات مهمة
- التثبيت والعمل بدون إنترنت يحتاجان تشغيل اللعبة عبر **HTTPS** مثل GitHub Pages أو `localhost`.
- إذا فتحت `index.html` مباشرة من الجهاز فلن تعمل ميزات PWA كاملة.
- بعد أول تشغيل ناجح على الإنترنت، سيُخزَّن التطبيق محليًا ويمكن تشغيله أوفلاين.
- التحكم البديل باللمس/الفأرة متاح إذا لم تعمل الكاميرا.

## ما تم إصلاحه
- استبدال الـ manifest المضمّن بملف حقيقي.
- إضافة Service Worker حقيقي.
- إضافة زر تثبيت داخل التطبيق.
- دعم GitHub Pages بالمسارات النسبية.
- الاحتفاظ بالإعدادات والنتائج محليًا.
- تحسين السلاسة ووضع السبورة ووضع المعلم.
